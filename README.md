# BlogPage - React Vite Project

## 🚀 Overview
BlogPage is a modern React-based single-page application built using Vite. It includes multiple sections like Home, About, Skills, Contact, and Footer, structured using reusable components and React Router for navigation.

---

## 🛠️ Tech Stack
- React.js
- Vite
- React Router DOM
- JavaScript (ES6+)
- CSS

---

## 📦 Installation & Setup

### 1. Clone the repository
```bash
git clone https://github.com/your-username/blogpage.git
```
### 2. Install dependencies
```bash

```
###3. Run development server
